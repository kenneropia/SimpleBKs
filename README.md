 
### Running Project

To run the project, you would use the following command:
`yarn run dev`

### Import Data

```
  mongoimport.exe --uri "{DB_URL}" --db simpleBKs --collection {COLLECTION_NAME} --type csv --headerline --file {CSV_FILE_PATH}
  
```
